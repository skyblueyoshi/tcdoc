# 基本JSON类型

## 脚本JSON格式

```text
{
    "path": <lua文件路径>,
    "requires": [
        [ <模块名称>, <lua模块文件路径> ],
        ...
    ]
}
```

| 标志 | 类型 | 默认值 | 描述 |
| :--- | :---: | :---: | :--- |
| path | 字符串 | **必填** | 所执行脚本的lua文件路径。 |
| requires | 列表 | 空 | 当前lua文件所使用的所有外部模块。 |
| &lt;模块名称&gt; | 字符串 | **必填** | 模块名称。lua中请使用`require(模块名称)`来加载指定模块。 |
| &lt;lua模块文件路径&gt; | 字符串 | **必填** | 模块对应的lua文件路径。 |
