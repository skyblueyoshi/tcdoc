# 音效表（sounds.json）

## JSON格式

```text
[
    {
        <音效ID名称>: {
            "tcResource": <布尔值>,
            "path": <音效文件路径>
        }
    },
    ...
]
```

| 标志 | 类型 | 默认值 | 描述 |
| :--- | :---: | :---: | :--- |
| &lt;音效ID名称&gt; | 字符串 | **必填** | 注册到游戏中的音效ID名称。 |
| tcResource | 布尔值 | false | 音效的绝对路径是否使用游戏assets主文件夹。若为false，则使用模组文件夹。 |
| path | 字符串 | **必填** | 音效文件路径。 |
