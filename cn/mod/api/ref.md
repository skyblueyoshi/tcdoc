# 引用API

## BlockEntityRef

### 类成员属性

| 属性 | 类型 | 描述 |
| :--- | :---: | :--- |
| BlockEntityRef.ref | BlockEntity/nil | 返回引用的方块实体对象，若对象不存在或为空引用则返回nil。 |

### 类成员函数

| 函数 | 返回值 | 描述 |
| :--- | :---: | :--- |
| BlockEntityRef:SetRef\(BlockEntity e\) | void | 设置当前引用指向指定方块实体，传入nil表示空引用。 |
