# Attack

## 属性 <a href="#attack" id="attack"></a>

|                                                                                                       |
| ----------------------------------------------------------------------------------------------------- |
| <p><strong>int attack</strong><br>攻击值。</p>                                                            |
| <p><strong>int knockBack</strong><br>攻击的击退值。</p>                                                      |
| <p><strong>int crit</strong><br>攻击的百分暴击率。1-100表示1-100%的概率产生双倍暴击伤害，大于100表示总是产生双倍暴击伤害，小于1表示不产生暴击伤害。</p> |
